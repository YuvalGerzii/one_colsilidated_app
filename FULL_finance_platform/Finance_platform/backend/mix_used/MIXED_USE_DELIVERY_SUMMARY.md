## üöÄ HOW TO USE THIS MODEL

### Step 1: Quick Start (15 Minutes)

1. **Open** `Mixed_Use_Model_v1.0.xlsx`
2. **Go to** Inputs sheet (Sheet 2)
3. **Enter Property Info**:
   - Property name and location
   - Total building SF (100K-1M SF)
   - Number of floors (6-60+)
   - Asset class (A/B/C)

4. **Set Space Allocation** (Rows 18-24):
   - Multifamily: ___%
   - Office: ___%
   - Retail: ___%
   - Hotel: ___%
   - Restaurant: ___%
   - **TOTAL: Must equal 100%**

5. **Input Assumptions** for each component:
   - Multifamily: Rents, occupancy, OpEx
   - Office: Rent/SF, load factor, occupancy
   - Retail: Base + % rent, sales/SF
   - Hotel: ADR, occupancy, F&B revenue
   - Restaurant: Rent, sales/SF, % rent

6. **Development Budget**:
   - Land cost
   - Hard costs/SF ($350-550)
   - Soft costs (15-20% of hard)
   - FF&E by component
   - Developer fee & contingency

7. **Financing**:
   - LTC: 60-70%
   - Interest rate: 6-8%
   - Term: 30 years

8. **Exit Assumptions**:
   - Hold period: 7-10 years
   - Exit cap rates by component
   - Selling costs: 2-3%

### Step 2: Review Scenarios (10 Minutes)

1. **Go to** Allocation Scenarios sheet (Sheet 3)
2. **Review** 8 pre-built strategies
3. **Identify** which scenarios fit your market
4. **Note** IRR, NOI, and metrics for each

### Step 3: Optimize Allocation (10 Minutes)

1. **Scroll down** to Custom Optimizer section
2. **Enter your percentages** (must total 100%)
3. **Review** calculated metrics:
   - Allocated SF
   - Est. NOI/SF
   - Total NOI
   - IRR impact

### Step 4: Validate Results (15 Minutes)

1. **Executive Summary** sheet - Check key metrics
2. **Returns Analysis** sheet - Verify IRR (target 18-25%)
3. **Consolidated Pro Forma** - Review 10-year NOI
4. **Sensitivity Analysis** - Test downside scenarios

### Step 5: Present to Stakeholders

1. **Use** Executive Summary as one-pager
2. **Print/PDF** key sheets for presentation
3. **Highlight** optimal allocation scenario
4. **Show** sensitivity analysis for risk discussion
