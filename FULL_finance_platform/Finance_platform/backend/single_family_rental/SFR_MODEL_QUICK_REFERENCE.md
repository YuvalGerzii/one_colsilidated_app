## üî¢ USEFUL FORMULAS

### Monthly Payment (Principal + Interest)
```
=PMT(rate/12, term*12, -loan_amount)

Example:
=PMT(0.075/12, 30*12, -217500)
= $1,531/month
```

### Future Value (With Appreciation)
```
=PV * (1 + rate)^years

Example:
=$290,000 * (1.04)^10
= $429,140
```

### Cap Rate
```
=NOI / Property_Value

Example:
=$15,534 / $204,050
= 7.6%
```

### Cash-on-Cash Return
```
=Annual_Cash_Flow / Initial_Investment

Example:
=$2,469 / $72,525
= 3.4% (Year 1 only)
```

### Debt Service Coverage Ratio
```
=NOI / Annual_Debt_Service

Example:
=$15,534 / $13,065
= 1.19x
```

## ‚ö†Ô∏è COMMON MISTAKES TO AVOID

### Input Errors
‚ùå Using asking rent (instead of market rent)  
‚ùå Forgetting to include HOA fees  
‚ùå Underestimating renovation costs  
‚ùå No contingency reserve (always add 10%)  

### Analysis Errors
‚ùå Ignoring vacancy (always assume 5%+)  
‚ùå Forgetting CapEx reserve ($100-200/mo)  
‚ùå Not stress-testing (check worst case)  
‚ùå Assuming appreciation (base case = 0%)  

### Execution Errors
‚ùå Overleveraging (keep DSCR ‚â•1.25x)  
‚ùå No property manager (hire at 5+ units)  
‚ùå Inadequate reserves (6+ months minimum)  
‚ùå Wrong market (research neighborhood trends)

## üéØ EXIT STRATEGY QUICK GUIDE

### Choose Flip If:
- Strong buyer demand
- Need capital for next deal (< 6 months)
- Simple exit preferred
- Market timing favorable

### Choose BRRRR If:
- Want to scale portfolio quickly
- Can recycle capital every 9-12 months
- Accept break-even or slight negative CF
- Building rental portfolio

### Choose Hold 10 Years If:
- 10-year IRR > 15%
- Positive cash flow Year 1
- Tax-efficient exit desired
- Don't need capital back soon

### Choose Hold Forever If:
- Excellent cash flow (>$500/mo)
- Long-term passive income goal
- Don't need capital back
- Legacy wealth building
