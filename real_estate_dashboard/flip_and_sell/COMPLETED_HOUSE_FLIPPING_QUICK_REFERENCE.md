## üî¢ KEY FORMULAS

### ROI (Return on Investment)
```
ROI = Gross Profit / Cash Invested
```
**Target:** 20%+ (adjust by market)

### Profit Margin
```
Profit Margin = Gross Profit / ARV
```
**Target:** 15%+ of ARV

### All-In Cost
```
All-In = Purchase + Closing + Reno + Holding + Finance Costs
```

### Gross Profit
```
Gross Profit = ARV - Selling Costs - All-In Cost
```

## üìà ADJUSTMENT FACTORS

### 70% Rule by Market
- **Hot Markets:** 60-65% (competitive)
- **Moderate:** 70% (standard)
- **Slow Markets:** 75-80% (less competition)

### By Property Price
- **<$200k:** Use 65% (less margin)
- **$200k-$500k:** Use 70% (standard)
- **$500k+:** Can use 75% (higher $ profit)
