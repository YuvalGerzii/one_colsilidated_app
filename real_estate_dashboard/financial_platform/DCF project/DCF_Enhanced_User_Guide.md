## üí° Key Concepts

### Terminal Value
- Represents value after projection period
- Typically 60-80% of total value
- Two methods: Gordon Growth or Exit Multiple
- **Most critical assumption** - validate carefully!

### WACC (Discount Rate)
- Weighted cost of equity and debt
- Typical range: 6-12%
- 1% change = 15-20% valuation impact
- Higher risk = higher WACC = lower value

### Sensitivity Analysis
- Tests multiple assumption combinations
- Shows range of potential values
- Identifies key value drivers
- **Use this before finalizing valuation!**

### Scenario Analysis
- Bull/Base/Bear cases
- Probability-weighted outcome
- Better than single point estimate
- Professional investors use this approach

## üéØ Best Practices

### 1. Start Conservative
Use conservative assumptions initially. Better to be pleasantly surprised.

### 2. Validate with History
Compare your projections to historical performance (Historicals sheet).

### 3. Multiple Methods
Use DCF + Trading Comps + Transactions for triangulation.

### 4. Sensitivity Test
Review sensitivity tables - understand what drives value.

### 5. Terminal Value Focus
Most value comes from terminal value - spend time validating this.

### 6. WACC Precision
A 1% WACC change = 15-20% valuation change. Get this right!

### 7. Scenario Planning
Don't rely on single estimate. Model Bull/Base/Bear cases.
