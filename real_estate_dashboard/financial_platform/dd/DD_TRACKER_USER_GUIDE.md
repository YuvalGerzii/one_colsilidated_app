### 2. **Master Tracker** üéØ (YOUR COMMAND CENTER)
**Purpose:** Central hub for all DD requests and items

This is your most important sheet. Everything flows through here.

**Column Guide:**

| Column | Field | Input Type | Purpose |
|--------|-------|------------|---------|
| A | ID | Auto | Unique identifier (1, 2, 3...) |
| B | Workstream | **INPUT** | Financial DD, Legal DD, Commercial DD, etc. |
| C | Category | **INPUT** | Sub-category within workstream |
| D | Item Description | **INPUT** | What you're requesting |
| E | Sub-Category | **INPUT** | Additional detail if needed |
| F | Responsible Party | **INPUT** | Who will provide (Internal/External/Target) |
| G | Assigned To | **INPUT** | Specific person's name |
| H | Status | **INPUT** | Not Started / Requested / In Progress / Received / Under Review / Complete / N/A |
| I | Priority | **INPUT** | Critical / High / Medium / Low |
| J | Target Date | **INPUT** | When you need it by |
| K | Date Completed | **INPUT** | When finished |
| L | Document Location | **INPUT** | VDR folder or file path |
| M | Red Flag | **INPUT** | Yes/No - is this a critical issue? |
| N | Follow-Up Required | **INPUT** | Yes/No - needs more action? |
| O | Risk Rating | **INPUT** | High/Medium/Low |
| P | Value Impact ($M) | **INPUT** | Financial impact if material |
| Q | Notes/Comments | **INPUT** | Any additional context |

**Status Definitions:**
- **Not Started:** Identified but not yet requested
- **Requested:** Formal request sent
- **In Progress:** Target company preparing
- **Received:** Documents received
- **Under Review:** Team analyzing
- **Complete:** Fully reviewed and closed
- **N/A:** Not applicable

**Priority Definitions:**
- **Critical:** Deal-breaker, immediate attention required
- **High:** Material to valuation, address within 3-5 days
- **Medium:** Important, address within 1-2 weeks  
- **Low:** Nice to have, address as time permits

**Best Practices:**
1. Add new items daily as DD progresses
2. Update Status field at least daily
3. Mark Red Flag = "Yes" for any material concerns
4. Be specific in descriptions ("2023 audited financial statements" not just "financials")
5. Always assign an owner and target date
